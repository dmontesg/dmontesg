## Codes I contributed:  

**Stellar kinematics**:

* [***SteParKin***](https://github.com/dmontesg/SteParKin/) python code to determine Galactic-space velocity components (U, V, W), evaluate their membership to young kinematic moving groups and associations and assign their stellar populations.
References: 
Montes et al. [2001MNRAS.328...45M](http://ui.adsabs.harvard.edu/abs/2001MNRAS.328...45M/abstract) and Cort√©s-Contreras et al. [2024A&A...692A.206C](https://ui.adsabs.harvard.edu/abs/2024A%26A...692A.206C/abstract)

**Chromospheric activity**:

* [***iSTARMOD***](https://github.com/flabarga/iSTARMOD) python code implementing Spectral Subtraction Technique to study chromospheric activity in Cool Stars.
Reference: 
*"iSTARMOD: A Python Code to Quantify Chromospheric Activity by Using the Spectral Subtraction Technique"*,
Labarga & Montes
[2026AJ....171...15L](https://ui.adsabs.harvard.edu/abs/2026AJ....171...15L/abstract), 
[2025ascl.soft12002L](https://ascl.net/2512.002), 
[0.5281/zenodo.17329153](https://zenodo.org/records/17329154)

**Stellar parameters**

* [***STEPAR***](https://github.com/hmtabernero/StePar): an automatic code to infer stellar atmospheric parameters. References:
Tabernero, H. M.; Marfil, E.; Montes, D. et al., [2019A&A...628A.131T](https://ui.adsabs.harvard.edu/abs/2019A%26A...628A.131T/abstract), [ascl.net/1907.018](http://ascl.net/1907.018)

* [***STEPARSYN***](https://github.com/hmtabernero/SteParSyn): A Bayesian code to infer stellar atmospheric parameters using spectral synthesis. References:
Tabernero, H. M.; Marfil, E.; Montes, D., et al. [2022A&A...657A..66T](https://ui.adsabs.harvard.edu/abs/2022A%26A...657A..66T/abstract), [ascl.net/2111.016](https://ascl.net/2111.016)

**Metallicity of M dwarfs**

* [***METaMorPHosis***](https://github.com/chrduque/metamorphosis): METallicity for M dwarfs using PHotometry. References:
Duque-Arribas, Christian; Montes, David; et al. [2022zndo...7428860D](https://ui.adsabs.harvard.edu/abs/2022zndo...7428860D/abstract),
*Photometric Calibrations of M-dwarf Metallicity with Markov Chain Monte Carlo and Bayesian Inference* 
Duque-Arribas, Christian; Montes, David; et al. [2023ApJ...944..106D](https://ui.adsabs.harvard.edu/abs/2023ApJ...944..106D/abstract)
